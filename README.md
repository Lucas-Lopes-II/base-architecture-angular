# Base Architecture Angular

## Installation

> To download and run the project you need to have git and node installed.

```bash
$ npm install
```

## Running the app

```bash
$ npm start
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
